About Flipping99

Flipping99 began as a personal journey — I was using physical flashcards to memorize the 99 Names of Allah and wanted something more elegant, portable, and interactive. So I built it: my first mobile app, designed to help others learn and reflect on these names anywhere.

This project pushed me to learn React Native, manage state and storage, and publish to the Play Store — handling everything from code to design to deployment. It’s not just an app; it’s a milestone in my growth as a developer and a way to share something meaningful with the world.
